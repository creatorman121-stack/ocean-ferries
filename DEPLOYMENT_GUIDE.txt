OceanJet PortDesk V95 GitHub Pages Deployment

1. Extract this ZIP.
2. Upload every file and folder to the repository root.
3. Confirm index.html is in the root, not inside another folder.
4. Commit and wait for GitHub Pages to publish.
5. Open the GitHub Pages URL.
6. Hard refresh once after deployment.

For future fare/advisory/module updates:
- Edit online-config.json in GitHub.
- The app fetches it on load and falls back to local cached/bundled data if offline.

Do not replace app.js with older versions and do not delete data.js, online_config.js, or v95_online_modules.js.
